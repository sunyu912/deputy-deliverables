# Deputy Deliverables

This repository is **auto-managed by [Deputy](https://github.com/sunyu912/my-todo-handler)**.

Each handled TODO gets its own folder here containing whatever files the task
produced (pages, reports, images, data, etc.). Deputy's local agent commits and
pushes those files automatically, and links back to them from the TODO.

GitHub Pages is enabled, so any folder with an `index.html` is viewable live at:

```
https://sunyu912.github.io/deputy-deliverables/<folder>/
```

Don't edit this repo by hand while the agent is running — it pulls before each
commit, but manual edits may conflict.
